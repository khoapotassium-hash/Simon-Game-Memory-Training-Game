# Simon Game - RISC-V Assembly Project
This project is a memory-training "Simon Game" developed using RISC-V Assembly language. It is designed to run on the RARS (RISC-V Assembler and Runtime Simulator) and was created for the Computer Architecture Lab (CA LAB) at Hanoi University of Science and Technology (HUST).

## Game Overview
Simon Game is a classic memory game where the player must replicate a sequence of signals generated by the system. The sequence length increases after each successful round, challenging the player's short-term memory.

### Technical Features
- Graphical Interface: Utilizes the RARS Bitmap Display.
- User Input: Interactive real-time control via the Keyboard and Display MMIO Simulator.
- Logic: Implements dynamic array management and random sequence generation in Assembly.

## Instructions for Execution

1. Clone the Repository:
   git clone https://github.com/khoapotassium-hash/Simon-Game-Memory-Training-Game.git
   cd Simon-Game-Memory-Training-Game

2. Launch the Simulator:
   Execute the RARS simulator located in the Environment directory:
   java -jar Environment/rars1_6.jar

3. Configure Tools:
   In the RARS "Tools" menu, initialize the following:
   - Bitmap Display:
     - Unit Width/Height: 8
     - Display Width/Height: 512
     - Base Address: 0x10008000 (heap)
   - Keyboard and Display MMIO Simulator.
   Note: Ensure both tools are "Connected to Program".

4. Run the Program:
   - Open index.asm.
   - Press F3 (Assemble).
   - Press F5 (Run).
